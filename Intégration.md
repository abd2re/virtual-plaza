# Intégration
Autre notation de $\int f(x)\,dx$=::$F(x)$
<!--SR:!2024-01-16,3,250-->
Si $g(x)$ = $u$ alors $g'(x)\,dx$=::$du$
<!--SR:!2024-01-16,1,227-->
## Identités
### Fondamentales
- $\int a\,dx=$::$ax+C$
<!--SR:!2024-01-16,3,250-->
- $\int x^{n}\,dx=$::$\frac{x^{n+1}}{n+1}+C;\;n\neq-1$
<!--SR:!2024-01-16,3,250-->
- $\int x^{-n}\,dx=\int \frac{1}{x^{n}}\,dx=$::$\frac{x^{1-n}}{1-n}+C;\;n\neq1$
<!--SR:!2024-01-16,3,250-->
- $\int x^{1/n}\,dx=\int \sqrt{x^{n}}\,dx=$::$\frac{nx^{\frac{n+1}{n}}}{n+1}+C;\;n\neq-1$
<!--SR:!2024-01-16,3,250-->
- $\int x^{-1}\,dx=\int \frac{1}{x}\,dx=$::$\ln|x|+C$
<!--SR:!2024-01-16,3,250-->
- $\int a^{x}\,dx=$::$\frac{a^{x}}{\ln(a)}+C$
<!--SR:!2024-01-16,1,190-->
- $\int e^{x}\,dx=$::$e^{x}+C$
<!--SR:!2024-01-16,3,250-->
- $\int \log_{a}(x)\,dx=$::$\frac{x}{\ln a}(\ln x-1)+C$
<!--SR:!2024-01-16,1,190-->
- $\int \ln x\,dx=$::$x(\ln x -1)+C$
<!--SR:!2024-01-16,3,250-->
- $\int \sin x\,dx=$::$-\cos x +C$
<!--SR:!2024-01-16,3,250-->
- $\int \cos x\,dx=$::$\sin x +C$
<!--SR:!2024-01-16,3,250-->
- $\int \sec^{2} x\,dx=$::$\tan x +C$
<!--SR:!2024-01-16,3,250-->

### Opérations
- $\int af(x)\,dx=$::$aF(x)$
<!--SR:!2024-01-16,3,250-->
- $\int f(x)\pm g(x)\,=$::$F(x)\pm G(x)$
<!--SR:!2024-01-16,3,250-->
- $\int f(x)g(x)\,dx=$::$F(x)g(x)-\int F(x)g'(x)\,dx$
